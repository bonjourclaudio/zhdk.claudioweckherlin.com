hungry!

2024 personal project web app
Project Description
I have developed a web application using Angular and Google Vision, designed to capture ingredients from a fridge, pantry, etc., and then provide recipes that can be prepared with the available ingredients.

The inspiration for this project stemmed from the common issue of browsing appealing recipes, only to discover missing ingredients in my fridge. "hungry" exclusively displays recipes that can be feasibly cooked.

The project's current scope is a proof of concept; it is not yet complete, and there are features that could be implemented in the future.
Implementation
For the sake of simplicity, I've opted to create a web application instead of a mobile app. Leveraging my extensive experience with the Angular framework, I selected this technology.
To match ingredients, I utilized the publicly available Google Vision API. This API allows for a straightforward submission of an image to their endpoint, which then returns x amount of matches and their corresponding confidence scores. As the API may return items that aren't necessarily foods, I match the obtained items with the ingredients from my JSON recipe dataset.
